06 / iteration boundaries

The loop grows sideways, not inward.

Scribectl permits new source material, independent fill variants, and new review passes. It does not permit an agent to revise its own output autonomously from another agent's feedback.

Breadth is allowed

Several independent proposals from the same frozen source may land side by side for human judgment.

Feedback is consumed manually

A review reports. The writer decides, reworks the draft, and requests another bounded pass if needed.

Canon is a one-way gate

Only a checkbox verdict and ratify --sweep move a claim from quarantine into a ratified node and ledger.
